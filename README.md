## 📌 Project Overview
This project collects real-time air quality data (AQI) from APIs, stores it in a cloud database, and visualizes it through an interactive dashboard.

## ⚙️ Features
- 🌍 Real-time AQI monitoring
- 📊 Historical trend analysis
- 🗺 Interactive AQI map
- 🔄 Automated data pipeline

## 🛠 Tech Stack
- Python
- Streamlit
- Supabase (PostgreSQL)
- Plotly
- GitHub Actions
